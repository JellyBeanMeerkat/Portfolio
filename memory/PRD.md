# UX Portfolio Website - Sharisse Cole

## Problem Statement
Build a UX portfolio website for Sharisse Cole, a UX Researcher & Designer. The site showcases her work, skills, and provides contact information.

## Architecture
- **Frontend-only** React app with React Router, Tailwind CSS, Shadcn UI, lucide-react icons
- **Font:** Cabinet Grotesk (Google Fonts)
- **Embedding:** Microsoft Office Online Viewer for .pptx presentations
- **Pages:** Home, Portfolio, About, Contact, Case Study (Hotel Booking)

## Completed Features
- Multi-page React app with client-side routing
- Custom hero section (60/40 split) with arched portrait frame
- Fixed header with navigation and social media icons (LinkedIn, Email)
- Portfolio page with project cards and "Coming Soon" placeholder
- Detailed Case Study page with embedded PowerPoint presentations, screenshots, external links (Figma, Miro)
- About page with skills, tools, and decorative presenting-figure image
- Contact page with mailto form and contact info
- All iterative design and content changes applied (Feb 2026)
